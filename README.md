# Number Quest

A simple terminal-based number guessing game written in Python.

## How to run

```bash
python3 game.py
```

## Gameplay

- Pick a difficulty (Easy, Medium, or Hard).
- Guess the hidden number within the attempt limit.
- Get hints (`higher` / `lower`) after each wrong guess.
- Earn points for guessing in fewer attempts.
- Play as many rounds as you want, with session stats tracked.
