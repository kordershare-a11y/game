# game

A small terminal version of 2048 written in Python.

## Run it

```bash
python3 game2048.py
```

## Controls

- `W` or `up` - move up
- `A` or `left` - move left
- `S` or `down` - move down
- `D` or `right` - move right
- `N` - start a new game
- `Q` - quit

## Notes

- No third-party packages are required.
- Reach the 2048 tile to win, then keep playing for a higher score if you want.
