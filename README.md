# game

A simple terminal Tic-Tac-Toe game written in Python.

## Run

```bash
python3 game.py
```

## How to play

1. Choose your symbol (`X` or `O`).
2. Choose a difficulty (`easy`, `medium`, or `hard`).
3. Enter the number of an open square to place your move.
4. Play additional rounds and track cumulative score.
